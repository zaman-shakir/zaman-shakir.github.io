Sticky notes (the FAQ).

→ five questions, answered

What does a DanDomain → Shopify migration cost?

Quoted per project after a paid discovery call. Typical catalog of 1–10k SKUs with two integrations and a redirect map under 5,000 URLs runs €8,000–€18,000 ex VAT. No hourly billing — fixed scope, fixed price.

How do you preserve SEO equity across a platform change?

Three things on the same day: a complete 301 redirect map matched URL-for-URL, the new shop's structured data matched to the old schema, and a crawl diff run within 48h of cutover. Most "lost ranking" stories come from skipping one of these.

Can you also build on WordPress?

Yes. WordPress + WooCommerce is about a third of the practice. Same workflow. WP wins when content is the product, full data ownership matters, or the catalog model breaks Shopify's variant ceiling.

How fast can a WP site actually load?

Most WP sites I ship hit under 1.0s First Contentful Paint on a cold cache. Page cache + object cache + edge fragment cache + OPcache + critical CSS + a strict plugin diet. Speed is a budget, enforced in CI.

Will I be one client among many?

No. Capacity capped at two simultaneous migrations. You work with me directly, weekly demos, single-thread questions. You own the code and the runbook when it ships.
